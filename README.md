dotfiles
========

My dotfiles.

## Dependencies
* `zsh`
* `vim`
* `python-virtualenvwrapper`
* `ctags` for vim plugins
* `xclip` for clipboard aliases

Change your shell to zsh with the `chsh` command.

## Installation
```
cd ~
git clone git@github.com:aairey/dotfiles.git .dotfiles
bash .dotfiles/install.sh
```
